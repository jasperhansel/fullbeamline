The cairo driver

The cairo driver uses the Pango/Cairo subset of the GTK+ suite of libraries to produce plots in the PDF, PostScript, EPS (Encapsulated PostScript), PNG, and SVG file formats. Text and graphics are antialised, font selection is done automatically for any given unicode glyph requested using the recommended fontconfig approach, and complex text layout is used to render text. As a result this driver is considered to be one of our two best (our qt driver described in the section called “The qt driver” is the other) file device drivers.
