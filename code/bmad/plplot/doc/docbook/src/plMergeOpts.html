plMergeOpts: Merge use option table into internal info structure.

PLINT plMergeOpts (options,
 name,
 notes);
 

Merges in a set of user-supplied command line options with the internal options table. This allows such options to be used along with the built-in PLplot options to set device driver, output file etc. See plparseopts for details of how to parse these options in a program.

options (PLOptionTable *, input)

User option table to merge.

name (PLCHAR_VECTOR, input)

Label to preface the options in the program help.

notes (PLCHAR_VECTOR *, input)

A vector of character strings which appear after the options in the program help.

This function is used in C examples 1, 8, 16, 20 and 21.
