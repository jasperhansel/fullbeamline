plmesh: Plot surface mesh for Fortran 95

plmesh (x,
 y,
 z,
 nx,
 ny,
 opt,
 mx);
 

When called from Fortran 95, this routine has the same effect as when invoked from C. The interpretation of all parameters (see plmesh) is also the same except there is an additional parameter given by:

mx (PLINT, input)

Length of array in x direction, for plotting subarrays.
