plSetUsage: Set the ascii character strings used in usage and syntax messages.

plSetUsage (program_string,
 usage_string);
 

Sets the program string and usage string displayed by the command line help option (-h) and by plOptUsage.

program_string (PLCHAR_VECTOR, input)

An ascii character string to appear as the name of program.

usage_string (PLCHAR_VECTOR, input)

An ascii character string to appear as the usage text.

This function is not used in any of our C examples.
