The purpose of this chapter is to document the PLplot C/C++ functions that are currently not part of the common API, either because they are C/C++ specific utility functions (e.g. plAlloc2dGrid, plFree2dGrid) or because they are not yet implemented in other languages (e.g., plMergeOpts). Some of these functions are used in the C or C++ examples and may be helpful to other C/C++ users of PLplot.

This chapter also documents some of the C/C++ structures defined by PLplot and used by these functions.
