19  This header file contains the lookup tables used for converting between
20  hershey and unicode. It is an automatically generated file, so please do
21  not edit it directly. Make any changes to plhershey-unicode.csv, then use
22  plhershey-unicode-gen.c to recreate this header file.
23 
24  plhershey-unicode.csv consists of three fields: the first field is the
25  hershey code, and is in decimal; the second is the unicode value, and is
26  in hex; and the final field is font index. There are five possible font
27  indices:
28  0 undefined/unknown
29  1 normal
30  2 roman
31  3 italic-roman
32  4 script
33 
34  Font indices are used for approximating the appearence of the original
35  hershey glyphs.
36 
37  Unicode values of 0x0000 signify unknowns.
38 
39 */
